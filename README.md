# Personal Website

Full-stack personal website built with React and Spring Boot.

## Features
- Contact form that sends messages to backend
- REST API built with Express.js
- MongoDB Atlas for data storage
- Frontend built with React

## Tech Stack
# Frontend
- React
- JavaScript
- HTML/CSS
# Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- REST APIs

## How to run locally

### Backend
```bash
cd contact-api
npm install
npm start
```
Make sure you have a .env file with:
```bash
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
### Frontend
```bash
cd my-website
npm install
npm start
```
